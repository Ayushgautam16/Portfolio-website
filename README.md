# Ayush Gautam Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. This project showcases Ayush Gautam's profile, skills, experience, certificates, resume, and social profiles in a polished single-page layout.

## 🚀 Live Demo
See the live portfolio on GitHub Pages or your hosting provider after deployment.

## ✨ Features
- Fully responsive single-page portfolio design
- Resume preview and certificate section
- Social profile links including X, YouTube, GitHub, LinkedIn, and Instagram
- Modern card UI with gradient effects and hover animations
- Clean contact and services sections
- Smooth horizontal social icon row with hidden scrollbar

## 🧩 Project Structure
- `index.html` — main portfolio page structure
- `style.css` — styles and responsive layout
- `script.js` — interactive behavior and animations
- `images/` — image assets for landing, projects, and social icons
- `netlify.toml` — optional deployment configuration

## 💻 Tech Stack
- HTML5
- CSS3
- JavaScript

## 🔧 How to Use
1. Clone the repository:
   ```bash
   
   git clone https://github.com/<your-username>/<repository-name>.git
   ```
   
2. Open `index.html` in a browser, or serve it locally with a web server.

## 📦 Deployment
This repository can be deployed using GitHub Pages or a static hosting platform such as Netlify or Vercel.

### GitHub Pages
1. Push the repository to GitHub.
2. In the repository settings, enable GitHub Pages from the `main` branch.
3. Set the root folder as the publishing source.

## 📝 Customize
- Update contact details in `index.html`
- Replace or add social icons in `images/social-icons/`
- Adjust colors and spacing in `style.css`
- Add more section cards or projects as needed

## 📞 Contact

If you want to make any further changes or add new sections, feel free to update the content in `index.html` or ask for help.
